## 📋 Descripción del Proyecto

Mural digital interactivo desarrollado como evidencia académica para el módulo de **Fundamentos de la Educación STEM**. Este proyecto presenta una infografía web moderna que explora la implementación de simuladores de maquinaria minera en Drummond, integrando principios STEM en entornos de formación profesional.

### 🎯 Características Principales

| Feature | Status | Descripción |
|---------|--------|-------------|
| **🎨 Diseño UX/UI Profesional** | ✅ Implementado | Paleta académica, tipografía Inter, layout responsivo |
| **📱 Experiencia Mobile-First** | ✅ Optimizado | Menú hamburguesa, touch-friendly, adaptable |
| **🌓 Modo Claro/Oscuro** | ✅ Persistente | Toggle con localStorage, accesibilidad visual |
| **🖼️ Galería Interactiva** | ✅ Modal Avanzado | Ampliación de imágenes, navegación intuitiva |
| **📊 Contenido Académico** | ✅ Completo | Rasgos metodológicos, análisis comparativo, referencias APA |
| **🎮 Interactividad STEM** | ✅ Integrado | Tooltips, tabs, animaciones scroll, íconos SVG |
| **📤 Exportación** | ✅ Funcional | Descarga PNG, copiado de referencias |

## 🚀 Instalación y Uso Local

### Prerrequisitos
- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor local para desarrollo (opcional)

### Pasos de Instalación
1. **Clonar o Descargar el Repositorio**
   ```bash
   git clone https://github.com/IngenieroDiego/MuralDigital-STEM-Drummond.git
   ```

2. **Estructura de Archivos**
   - Asegurar que todas las imágenes estén en la carpeta raíz
   - Verificar que los tres archivos principales estén presentes

3. **Ejecución Local**
   - Abrir `index.html` en cualquier navegador
   - Para desarrollo: usar servidor local como Live Server

### Personalización
- **Contenido**: Modificar texto directamente en HTML
- **Estilos**: Ajustar variables CSS en `:root`
- **Imágenes**: Reemplazar archivos en assets/ manteniendo nombres
